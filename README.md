# 動画一覧アプリ（Kotlin / Android）

## 概要
APIから動画一覧を取得し、リスト表示するAndroidアプリです。

## 使用技術
- Kotlin
- MVVMアーキテクチャ
- ViewModel + LiveData / StateFlow
- Retrofit + Room
- Coroutine
- Navigation Component
- Hilt（DI）
